# 🎮 프롬프트 아케이드 (Prompt Arcade) - 아크랩스

[![Aklabs Link](https://img.shields.io/badge/Aklabs-Link-orange?style=for-the-badge)](https://litt.ly/aklabs)

**프롬프트 아케이드 (Prompt Arcade)**는 최신 생성형 AI 기술과 오프라인 교구형 보드게임 설계 기법을 결합하여, 학생들이 직접 AI 게임 및 시나리오를 창작하고 실시간으로 배틀 세션을 진행할 수 있는 **크리에이티브 에듀테인먼트 플랫폼**입니다.

---

## 🚀 주요 기능 (Key Features)

### 1. 실시간 격리형 멀티룸 배틀 시스템 🎮
* **격리 세션 개설:** 호스트(선생님)가 고유 방 이름 및 **4자리 랜덤 코드**를 발급하여 독립된 세션을 즉석에서 개설할 수 있습니다.
* **자가 복구(Self-Healing) 시스템:** 방 입장 코드가 누락된 비정상/이전 게임 세션을 호스트 로딩 시 자동으로 실시간 감지하여 즉시 4자리 난수 코드를 부여하고 DB를 정규화하여 복원하는 자가 복구 아키텍처가 이식되었습니다.
* **동시성 오염 이중 방어막:** 여러 클라이언트가 동시에 세션을 동기화하면서 발생하는 데이터 중복 인서트 병목을 원천 방어하기 위해 **백엔드 권한 가드(호스트 전용 생성 제한)** 및 **프론트엔드 유니크 필터 렌더링**의 이중 방어막을 구축하여 동기화 신뢰성을 보장합니다.

### 2. AI 게임 기획 창작소 (AI GameCreatorGuide) 🛠️
* 컴퓨터가 없는 환경에서도 주변의 양면 메모 패드, 다면체 주사위, 카드 등의 실물 도구와 AI 프롬프트를 연계한 오프라인 RCTF(역할, 상황, 수행, 형식) 게임을 자유롭게 디자인할 수 있는 **창작 가이드 벤토 덱**을 지원합니다.
* AI 기반 기획 대장간 알고리즘이 탑재되어, 사용자가 입력한 핵심 테마와 RCTF 요소들을 조합한 풍부한 오프라인 게임 기획서를 즉석에서 자동 생성합니다.
* 생성된 기획서는 즉시 클립보드에 복사하거나 공유 피드(Community Feed)에 영구 기여할 수 있습니다.

### 3. 실시간 AI 프롬프트 피드 및 퀴즈 매칭 🕵️‍♂️
* 크리에이터들이 제작한 AI 프롬프트 설정값 및 게임 기획서들을 실시간으로 등재하고, 추천(좋아요)을 실시간으로 누를 수 있는 공유 커뮤니티 공간을 제공합니다.
* 동적 CRUD 에디터가 탑재되어 어드민 권한이 있는 관리자는 언제든지 아케이드 목록과 상세 소개서 페이지를 웹 UI 상에서 동적으로 수정하고 업데이트할 수 있습니다.

### 4. 아날로그 효과음 및 초고속 WebP 이미지 업로더 🔊
* **Web Audio API 오실레이터 효과음:** 네트워크 로딩 지연이 아예 없는 브라우저 네이티브 기술로 슬롯 회전 시 주파수가 변조되는 8비트 레트로 아케이드 효과음 및 멈추는 정확한 순간 딩동댕 팡파레 멜로디를 실시간 재생합니다.
* **클라이언트 사이드 WebP 초고속 압축:** 이미지 파일 선택 시 즉시 Canvas API를 활용해 해상도를 **가로 최대 1280px로 다운스케일링**하고 **WebP 포맷(80% 화질 압축)**으로 즉석 인코딩하여 수십MB 크기의 무거운 촬영 원본을 수십KB 수준의 깃털처럼 가벼운 초경량 크기로 백그라운드 압축합니다.
* **Supabase Storage 연동:** 압축 완료된 WebP Blob 데이터를 Supabase Storage (`rctf-images` 버킷)로 즉시 다이렉트 전송 및 퍼블릭 URL을 획득해 자동으로 카드 이미지 폼에 바인딩 연동해 줍니다. 버킷이 부재할 시 자동으로 Public 버킷을 백엔드에 사전 시딩해 생성하는 지능형 폴백 설계가 탑재되어 있습니다.
